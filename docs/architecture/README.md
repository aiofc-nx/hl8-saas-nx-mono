## 🎯 架构概览

HL8 SAAS 平台采用**混合架构模式**：

```
混合架构
├── Clean Architecture      # 分层架构基础
├── DDD                    # 领域驱动设计
│   └── Rich Domain Model  # 充血模型
├── CQRS                   # 命令查询分离
├── Event Sourcing         # 事件溯源
└── Event-Driven           # 事件驱动
```

---

## 🏗️ 架构分层

```
┌─────────────────────────────────────┐
│      Interface Layer (接口层)        │  ← HTTP/CLI/消息队列
├─────────────────────────────────────┤
│    Application Layer (应用层)        │  ← 用例、CQRS、事件处理
├─────────────────────────────────────┤
│      Domain Layer (领域层)           │  ← 实体、聚合根、值对象
├─────────────────────────────────────┤
│  Infrastructure Layer (基础设施层)   │  ← 数据库、缓存、消息队列
└─────────────────────────────────────┘
```
